Install MSW 2.x và generate service worker file.

## Requirements

1. Install MSW 2.x package
2. Generate service worker script
3. Verify installation

## Implementation Steps

### Step 1: Install MSW

```bash
cd apps/web
pnpm add -D msw@latest
```

### Step 2: Generate Service Worker

```bash
pnpm dlx msw init public/ --save
```

Lệnh này sẽ:
- Tạo `public/mockServiceWorker.js`
- Update `package.json` với `msw.workerDirectory`

### Step 3: Verify Installation

```bash
# Check msw version
pnpm list msw

# Check service worker file exists
ls -la public/mockServiceWorker.js
```

### Step 4: Create Mocks Directory Structure

```bash
mkdir -p src/mocks/handlers src/mocks/data
touch src/mocks/browser.ts
touch src/mocks/server.ts
touch src/mocks/handlers/index.ts
```
